# تطبيق محامي فون (Mohamy Phone) 📱💼

تطبيق أندرويد متكامل ومصمم خصيصاً للمحامين لإدارة مكاتبهم القانونية وحقائب القضايا والعملاء بخصوصية مطلقة وأمان تام (**Local-first & Offline-first** للبيانات القانونية).

---

## 🔍 نظرة عامة على المشروع (Project Overview)

**محامي فون** هو الحل التقني المثالي للمحامي العصري لربط وإدارة أعماله اليومية. يعمل التطبيق محلياً على نظام الأندرويد، حيث تُخزن كافة البيانات والمستندات والملفات المرفوعة بشكل آمن داخل المساحة الخاصة بالتطبيق على الهاتف الذكي.  
بيانات القضايا والملفات لا تُرسل للخادم؛ الاتصال الشبكي (إن فُعِّل) يُستخدم فقط لتنشيط الترخيص والتحقق الدوري منه.

* **بيئة تشغيل التطبيق:** Android (Kotlin & Jetpack Compose)
* **قاعدة البيانات المحلية:** Room SQLite Database
* **طبيعة الاتصال:** Local-first للبيانات القانونية، مع خيار اتصال بسيرفر ترخيص عند التفعيل.

---

## 🔑 نظام التفعيل

يتطلب التطبيق حساب ترخيص يتم إنشاؤه من لوحة الإدارة داخل `admin-server`.
في بيئة التطوير فقط (`BuildConfig.DEBUG`) يتوفر حساب اختبار محلي (`test / 123456`) كمسار طوارئ عند تعذر الوصول للخادم.

## ✨ الميزات الحالية المعتمدة (Current Supported Features)

1. **إدارة العملاء (CRUD):** إضافة جديد، تعديل تفاصيل، أرشفة، وحذف بيانات العملاء مع سياق قضاياهم.
2. **إدارة القضايا (CRUD):** تنظيم القضايا، رقم الدعوى، المحكمة، الخصم، وتحديث الملاحظات التراكمية والجلسات بشكل منظم.
3. **أجندة الجلسات (Sessions Tracker):** جدولة ومتابعة الجلسات القادمة والسابقة وقرارات المحكمة وربطها بالدعاوى.
4. **المهام والواجبات اليومية (To-Do Tasks):** إضافة مهام معينة بملف قضية وتأشيرها كمكتملة بعد الإنجاز لضمان عدم فوات المواعيد الرسمية.
5. **نظام أرشفة المستندات المتقدم (Advanced File Archival):**
   - استيراد الملفات بمختلف الصيغ (`TXT`, `PDF`, `DOCX`, `XLSX`, `JPG/PNG`).
   - الكشف الديناميكي التلقائي عن الاسم الأصلي للملف، حجمه (`formatFileSize`)، وامتداده، ونوع الـ `MIME`.
   - قراءة واستخراج النصوص تلقائياً لملفات النصوص مسطحة الهيكل `.txt`.
   - إتاحة واجهة "كشاف الفهرسة اليدوي" لإدخال متن أو كلمات مفتاحية للمستند لمقاومة ملفات PDF والملفات المصورة وضمان الوصول السريع لها.
   - إمكانية فتح وعرض الملفات تلقائياً عبر تطبيقات الهاتف الأصلية المثبتة ومشاركتها آلياً عبر نظام `FileProvider`.
6. **الباحث الموحد وكشاف المستندات (Unified Search Module):**
   - معالجة النصوص للمطابقة العربية المعقدة المقاومة للحركات والتشكيل (إزالة التشكيل، التطويل، توحيد الألف الأشكال أ/إ/آ وتوحيد الياء والى والمسافات).
   - تقسيم نطاق البحث محلياً لـ 4 خيارات مرنة: **بكل الملفات**، **بحقيبة قضية معينة**، **تحت اسم عميل تصفية**، أو **بمستند واحد معين**.
   - دعم التوليد الفوري للمذكرات بنقرة زر واحدة لحفظ المقتطف النصي المطابق كمذكرة ملحقة بملف الدعوى.
7. **المساعد الذكي المحلي (Local Smart Assistant):**
   - محرك محلي يعالج سياق وبيانات القضية لتلخيصها، جلب المستندات الناقصة، تذكير بالجلسات وإظهار قائمة المراجعة القانونية كلياً بشكل محلي 100% وبسرعة فائقة.
   - يتضمن تحذيراً وإخلاءً قانونياً صارماً:
     > *"⚠️ تنبيه: المساعد الذكي يساعد في تنظيم البيانات والبحث داخل الملفات وتجهيز القوالب، ولا يعتبر رأيًا قانونيًا نهائيًا."*
8. **النماذج القانونية وتوليد العقود (Templates & Generation):**
   - استعراض النماذج الجاهزة، تعبئة البنود التفاعلية وصياغة وتصدير العقود والدعاوى المكتملة بصيغة نصية قابلة للنسخ والتعديل والطباعة.
9. **النسخ الاحتياطي اليدوي (Local Backup & Restore):**
   - وظائف تصدير نسخة احتياطية من قاعدة بيانات التطبيق لقراءة واسترجاع الحالة بأمان تام.

## ⏳ الميزات المؤجلة للإصدارات القادمة (Delayed Features)

لكونه إصداراً محلياً خالصاً، تم إرجاء الميزات التالية للنسخ اللاحقة:
* تحسينات استخراج النصوص من صيغ غير `TXT` بشكل محلي.
* مزامنة اختيارية مستقبلية (غير مفعلة في الإصدار الحالي).
* توسعة نظام التراخيص المحلي.

---

## 🚫 محددات وثوابت المعمارية (Architectural Constraints)

* **لا توجد أي اتصالات إلزامية لبيانات القضايا:** القضايا والملفات تحفظ محلياً فقط.
* **لا يوجد Gemini/OpenAI/Firebase AI داخل التطبيق.**

## 🌐 لوحة الويب والإدارة

المشروع يتضمن خادماً إدارياً جاهزاً داخل المسار:
`admin-server`

ومن خلاله تستطيع:
* إنشاء العملاء (المحامين) وحسابات الدخول.
* إدارة التراخيص، الأجهزة، الإيقاف/الحظر، وتصفير ربط الأجهزة.
* ربط لوحة الويب على أي دومين وسيرفر عبر Nginx/PM2.

راجع دليل النشر الكامل:
`admin-server/README.md`

## 💻 كيفية التشغيل والمعاينة (How to Run in Android Studio)

1. افتح تطبيق **Android Studio (Ladybug or newer)**.
2. اختر **Open** وحدد المجلد الجذري للمشروع.
3. اسمح للأندرويد ستوديو بعمل مزامنة لـ Gradle (**Gradle Sync**) وتحميل الاعتماديات المحددة والمحدثة بملف `gradle/libs.versions.toml`.
4. لتشغيل التطبيق، قم بتوصيل هاتفك أو تشغيل المحاكي وانقر على زر **Run 'app'**.

### 🛠️ كيفية البناء اليدوي لنسخة الـ APK (Build Debug APK)

يمكنك بسهولة بناء نسخة الـ APK الجاهزة للتثبيت باستخدام منفذ الأوامر بمجلد المشروع:

#### لنظام لينكس / ماك (Linux / macOS):
```bash
# تصفية التنظيف الأساسي
./gradlew clean

# بناء ملف الأكواد ومستودع الـ APK
./gradlew :app:assembleDebug
```

#### لنظام ويندوز (Windows CMD / PowerShell):
```cmd
# تصفية وتنظيف الملفات المؤقتة
gradlew.bat clean

# بناء ملف الـ APK
gradlew.bat :app:assembleDebug
```

إذا كان مسار المشروع يحتوي أحرفاً غير لاتينية وظهر خطأ ترميز في `gradlew.bat`، استخدم نفس Gradle Wrapper مباشرة:

```cmd
java -classpath "gradle/wrapper/gradle-wrapper.jar" org.gradle.wrapper.GradleWrapperMain clean
java -classpath "gradle/wrapper/gradle-wrapper.jar" org.gradle.wrapper.GradleWrapperMain :app:assembleDebug
```

بعد انتهاء البناء بنجاح، ستجد ملف الـ APK بالموقع التالي جاهزاً للنقل والتثبيت:
`/app/build/outputs/apk/debug/app-debug.apk`

## 🛡️ الخصوصية والأمان (Trust & Local Sovereignty)

جميع ملفات الموكلين والمذكرات القانونية والبيانات المالية والرسوم تُحفظ بتطابق كامل وتام داخل تخزين التطبيق المنعزل عن بقية التطبيقات والمنصة. لا يتم تتبع الإحصاءات أو إرسال تقارير الأعطال لخوادم تجارية سعياً للحفاظ على أعلى درجات أسرار المحاماة والقوانين الوطنية والشرعية المنظمة لأعمال مكاتب الاستشارات القانونية.
